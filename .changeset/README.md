# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool that works
with multi-package repos, or single-package repos to help you version and publish your code. You can
find the full documentation for it [in the repository](https://github.com/changesets/changesets).

## Adding a changeset

To create a changeset, run `pnpm changeset` in the root of the repository. You'll be prompted to:

1. Select which packages have changed
2. Choose the semver bump type (major, minor, patch)
3. Write a summary of the changes

The changeset file will be committed with your PR. When the PR merges, the changesets/action bot will
open a "Version Packages" PR that bumps versions and updates changelogs.
